# 📊 Project Overview

Consumer360 is a **Retail Customer Analytics system** designed to analyze customer behavior using **RFM (Recency, Frequency, Monetary) segmentation**.

The project helps businesses:

* Identify **high-value customers**
* Detect **churn-risk customers**
* Improve **marketing campaigns**
* Understand **customer purchase patterns**

---

# 🧠 Key Analytics Techniques

* **RFM Segmentation** – Categorizes customers based on purchase behavior.
* **Cohort Analysis** – Tracks customer retention over time.
* **Market Basket Analysis** – Identifies products frequently bought together.

## Week 1 – Data Engineering

  * Fact Sales

  * Dim Product

## Week 2 – Analytics Logic

  * Champions
  * Loyal Customers
  * Potential Loyalists
  * Hibernating
  * Lost

**Output:** Customer segmentation dataset.
